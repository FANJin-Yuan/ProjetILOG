# Répertoire personnel ILOG
Ce répertoire personnel "ILOG" est créé ici à titre indicatif dans le dossier personnel de l'utilisateur par défaut.
Il est voué à contenir des informations sur le module, des prises de note éventuelles de la part de l'utilisateur etc.
Bien sûr, n'importe quel utilisateur peut créer des répertoires personnalisés pour tous les modules qu'il suit.
